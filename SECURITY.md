# Security and Privacy

This repository is intended to be public.

Do not commit:

- real tenant records
- production CSV exports
- staff records
- private SharePoint URLs
- Power Apps packages containing environment-specific details
- screenshots or videos showing personal data
- credentials, tokens, client secrets, or `.env` files

Use fake `example.com` email addresses and generic placeholder organisation names in documentation and samples.

If you discover sensitive data in the repository, remove it from the working tree immediately and rotate any exposed credentials. If sensitive data has already been pushed to GitHub, treat the repository history as exposed and follow GitHub's guidance for removing sensitive data from history.
